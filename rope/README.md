# Sigmoid

## 0x00 说明

RoPE基础版本，包含了RoPE在Llama的最小实现。

包含以下内容：

- [X] rope_f32_kernel
- [X] rope_f32x4_kernel(float4向量化版本)
- [X] PyTorch bindings


## 测试

```bash
# 只测试Ada架构 不指定默认编译所有架构 耗时较长: Volta, Ampere, Ada, Hopper, ...
export TORCH_CUDA_ARCH_LIST=Ada 
python3 rope.py
```
